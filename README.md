# SqueakMouse
A custom mouse case with holes for buttons, scroll wheel, screw holes and custom shape, made for the bambu labs mouse kit.
To use this case, insert the mouse PCB board wiht a battery into the case, align the screw holes and screw it in. Make sure the button and scroll wheel are aligned to the case.
I made this project to learn to use onshape, and build cases for custom pcbs and other devices.
